Glossary of HTML and CSS

American Standard Code for Information Exchange (ASCII)
A a 7-bit character code where each individual bit represents a unique character in the English alphabet.
The extended ASCII table — ASCII Code
ASCII.

character encoding
The process of assigning numbers to graphical characters allowing them to be stored, transmitted, and transformed electronically.
Character Encoding — Wikipedia
character entity reference
A short memorable name used in place of a numeric character reference for character entities that replace reserved characters in HTML.
HTML Entities — W3Schools
character set
A collection of characters and symbols representing every character that a browser is capable of rendering.
HTML Character Sets — W3Schools
client
A computer or other device that receives a service.
Client Definition — TechTerms
client-server network
A network design in which several clients access information from one or more servers.
Client-server architecture — Britannica
deprecation
The process by which obsolete features of a programming language are phased out.
Deprecated — MDN Web Docs
description list
A list containing a list of terms, each followed by its description (definition).
The Description List element — MDN Web Docs
doctype switching
The process by which browsers use the presence or absence of a doctype to decide which mode they should use to render a document.
Quirks Mode and Standards Mode — QuirksMode
empty element
An element that does not contain content.
Empty element — MDN Web Docs
Extensible Hypertext Markup Language (XHTML)
A stricter version of HTML4 created using XML, but which is now deprecated (obsolete).
HTML XHTML — W3Schools
Extensible Markup Language (XML)
A compact derivatie of SGML, used to define new markup languages.
What is XML? — W3Schools
file server
A network host device that provides storage space for saving and retrieving files.
File server — Computer Hope
host
Device on a network that can be used to share information and services.
What is a Host? — Webopedia
HTML5
The latest version of HTML, which supplanted XHTML 2 as the current Web document language.
HTML5 — MDN Web Docs
hypertext
A method of organization in which data sources are interconnected through a series of links or hyperlinks that users can activate to jump from one piece of information to another.
What is Hypertext? — W3C
Internet service provider (ISP)
An organization or company that provides Internet access.
What Is an Internet Service Provider? — Lifewire
ISO 8859-1 (Latin-1)
A character set that supports 255 characters and can be used by most languages that employ the Latin alphabet, including English, French, Spanish, and Italian.
ISO-8859-1 Character Set — W3Schools

Markup language — Britannica
