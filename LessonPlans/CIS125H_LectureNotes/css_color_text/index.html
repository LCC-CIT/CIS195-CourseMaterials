Types of Cascading Style Sheets

  • Inline
    Defined in an element. Apply to a specific XHTML element (tag).
  • Embedded
    Defined in the header. Apply to a whole web page
  • External
    Defined in a separate file. Can apply to multiple web pages
  • Imported
    Defined in a separate file. Can become part of another embedded or external CSS
