Configuring Color with CSS

  • Colors can be specified by number
    • RGB color codes represent a mix of three colors
      • Red
      • Green
      • Blue
    • Each pair of digits represents one color
      • #FF0000 = Red
      • #00FF00 = Green
      • #0000FF = Blue
    • Color schemes and numbers can be found in the center insert of the textbook
      or on many web sites like Color Jack
    • Example:
      <style type="text/css">
      body { background-color: #438897;
      color: #DCAD03;
      }
      </style>
  • Colors can be specified by name
    But, color numbers are preferred over color names
