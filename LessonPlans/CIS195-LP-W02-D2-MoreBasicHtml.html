More HTML Elements

  1. Nested lists
    Lists can be placed within lists. The right way to do it is to put the nested list inside a li element of the parent list. For example:

    <ul>
       <li>Deserts
         <ul>
             <li>Pie</li>
             <li>Cake</li>
            <li>Ice Cream</li>
          </ul>
      </li>
        <li>The rest of the list</li>
    </ul>

    This would look like this:

    • Deserts
      • Pie
      • Cake
      • Ice Cream
    • The rest of the list

  2. The hgroup element—deprecated
    This element is now obsolete—it isn't part of the HTML 5 specification. However, all the major browsers still recognize it.

    In our textbook, the author uses it to group a heading and a sub-heading.

    <header>
      <hgroup>
        <h3>This is a heading</h3>
       <h4>This is a sub-heading</h4>
      </hgroup>
    </header>

    Which looks like this:

    This is a heading

    This is a sub-heading

    The reccomended way to group them now is with just the header element. Note that this grouping doesn't change the appearance. It's just for organization and to facilitate styling (we'll learn about that later). Here's the same heading and sub-heading grouped inside just a header element:

    <header>
       <h3>This is a heading</h3>
      <h4>This is a sub-heading</h4>
    </header>

    Which looks the like this:

    This is a heading

    This is a sub-heading


  3. Link Element
    This element isn't for creating links to other pages. It is used to link style sheets (.css files) and other resources to your web page. Example:

    <link rel="stylesheet" href="MyStylesheet.css">
  • rel means relationship
  • href means http referenence (a filename or web address)
  • Read more at W3Schools

In-line vs. Block Elements

In-line elements arrange themselves horizontally on the page. Examples are:

  • <em>
  • <strong>
  • <img>

Block elements arrange themselves vertically on the page. Examples are:

  • headings (<h1>, <h2>, etc.)
  • <p>
  • <ul> and <ol>
