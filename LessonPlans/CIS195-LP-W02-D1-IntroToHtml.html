  2. Adding links to the css and JavaScript files provided by the author of the textbook 
    <script src="modernizr-1.5.js"></script>
<link href="jpsstyles.css" rel="stylesheet" type="text/css" />
    • Modernizr is designed to help you make HTML5 and CSS3 work more consistently across different browsers-- which was more of an issue when the textbook was published. It's not such an issue now. You can leave Modernizer out.
    • The CSS file will dramatically change the layout and styling of your page. It will center the image at the top, move the asside to the right side of the page, add background color, and more!
  3. FYI, <hgroup> is now deprecated It's OK to still use it, but it doesn't do much for you and it is no longer part of the official HTML5 specification.
