Nested Tables

One outer table containing 4 inner tables

  • Nesting
    • The outer table has four rows, but only the 1st and 3rd contain td elements
    • Each inner table is inside a td in a row of the outer table
  • Column headers
    • The outer table has a thead row containing four th elements
    • Each of the four td elements has a colspan of 2 so that it spans 2 th elements
  • Row headers
    • The outer table has four tr elements, each containing a th element
    • The td elements containing the inner tables have a rowspan of 2 so that they span 2 th elements
