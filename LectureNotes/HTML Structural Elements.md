### HTML Structural Elements 

- Non-semantic structural element

- - `<div> some other elements would go here </div>`

- These are *semantic structural elements* (also known as *semantic sectional elements*).

  ```html
  <header>
  </header>
  <section>
       <article>
       </article>
  </section>
  <aside>
  </aside>
  <footer>
  </footer>
  ```
