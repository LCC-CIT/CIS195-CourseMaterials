Centering Images in Figures

This example shows you how to center two different types of elements horizontally:
  • Inline elements
    • Images are in-line elements (as is text and other elements like anchors). In this example the images will be centered inside the figure elements.
    • Inline elements are centered with the text-align:center CSS property.
  • Block elements
    • Figures are block elements (as are paragraphs, headings, and others).
    • Block elements are centered by applying the CSS property margin: auto, but there are a few other things you need to do to make this work:
      • Set the block element you are centering to a fixed or fluid width. Using a percentage works nicely.
      • Since you are centering horizontally and block elements stack vertically, you need to float the block elements.
