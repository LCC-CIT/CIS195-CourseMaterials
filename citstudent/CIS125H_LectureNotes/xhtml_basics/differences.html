Main differences between HTML and XHTML


XHTML must be well-formed

  • Use lowercase
  • Use opening and closing tags
  • Close stand-alone tags with special syntax

XHTML must begin with an XML directive

XHTML requires a Document Type Definition (DTD)

  • XHTML 1.0 Transitional
  • XHTML 1.0 Strict
  • XHTML 1.0 Frameset
